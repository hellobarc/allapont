Distribución Wishart

Es una distribución para matrices simétricas, definidas positivas.

Sea \(\mathbf{X}_1,...,\mathbf{X}_n\) vectoreas aleatorios independientes con ditribución \(\mathcal{N}_p(0,\mathcal{\Sigma})\) que forman una matriz \(p \times n\) de datos \(\mathbf{X} = [ \mathbf{X}_1,...,\mathbf{X}_n] \).

Se afirma que una matrix \(\mathbf{M}_{p \times p}=\mathbf{X}\mathbf{X}'=\sum_{i=1}^{n}\mathbf{X}_i\mathbf{X}_{i}'\) tiene distribución Wishart.

Definición 1:

Una matriz \(\mathbf{M}_{p \times p}=\mathbf{X}\mathbf{X}'=\sum_{i=1}^{n}\mathbf{X}_i\mathbf{X}_{i}'\) tiene distribución Wishart con \(n\) grados de libertad y matriz de covarianza \(\mathcal{\Sigma}\) se denota por \( \mathbf{M} \sim \mathcal{W}_p (n,\mathcal{\Sigma}) \). Para \(n \geq p\). La función de densidad de probabilidad de \(\mathbf{M}\) es:

\[ f(\mathbf{M})=\frac{1}{ 2^{np/2}\Gamma_{p}(\frac{n}{2})|\mathcal{\Sigma}|^{n/2} }|\mathbf{M}|^{(n-p-1)/2 } \exp \left( -\frac{1}{2} \textrm{tr}(\mathcal{\Sigma}^{-1}\mathbf{M}) \right) \]
