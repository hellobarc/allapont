1. Número complejos - Ejercicio 7.1.a

Hallar un valor de \(\text{arg} (z)\) para \(z = \frac{-2}{1+\sqrt{3}i}\)

Argumento de un número complejo

El argumento de un número complejo \(z=x+iy\) está dado por la siguiente regla:

\( \text{arg} (z) = \begin{cases} \arctan (y/x) &\text{si } x > 0 \\ \arctan (y/x)+ \pi &\text{si } x < 0, y \geq 0 \\ \arctan (y/x)- \pi &\text{si } x < 0, y < 0 \\ \pi/2 &\text{si } x = 0, y > 0 \\ -\pi/2 &\text{si } x = 0, y < 0 \\ \text{indefinido} &\text{si } x = 0, y = 0 \\ \end{cases} \)

División por un número complejo

Para dividir por un número compejo, hay que multiplicar el numerador y el denominador por el conjugado del número complejo que se está dividiendo:

\( \begin{aligned} \displaystyle \frac{x+iy}{u+iv} &= \frac{x+iy}{u+iv} \frac{u-iv}{u-iv} \\ \end{aligned} \)

Por lo tanto se escribe la expresión como:

\( \begin{aligned} \displaystyle \frac{-2}{1+\sqrt{3}i} &= \frac{-2}{1+\sqrt{3}i} \frac{1-\sqrt{3}i}{1-\sqrt{3}i} \\ &= \frac{-2+2\sqrt{3}i}{1-\sqrt{3}i+\sqrt{3}i-(\sqrt{3}i)^2} \\ &= \frac{-2+2\sqrt{3}i}{1-\sqrt{3}^2 i^2} \\ &= \frac{-2+2\sqrt{3}i}{4} \\ &= \frac{2(-1+\sqrt{3}i)}{4} \\ &= \frac{-1+\sqrt{3}i}{2} \\ &= \underbrace{\frac{-1}{2}}_{x=\text{Re}(z)} + i\underbrace{\frac{\sqrt{3}}{2}}_{y=\text{Im}(z)} \\ \end{aligned} \)

Ahora se halla el argumento de \(\text{arg} (z)\), como \(x < 0, y \geq 0\), el argumento viene dado por: \( \displaystyle \arctan \Bigl( \frac{y}{x} \Bigl) +\pi \)

Entonces pude escribirse:

\( \begin{aligned} \displaystyle \arctan \Bigg( \frac{\frac{\sqrt{3}}{2}}{\frac{-1}{2}} \Bigg) +\pi &= \arctan ( -\sqrt{3} ) +\pi \\ &= -\frac{\pi}{3}+ \pi \\ &= \frac{2\pi}{3} \\ \end{aligned} \)

\( \therefore \text{arg}(z) = \displaystyle \frac{2\pi}{3} \)
